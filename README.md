# Docker - Selenium Grid - TestNG - Parallel Execution

This is a sample project for Docker - Selenium implementation with TestNG (Parallel Execution)

## Step 1:

docker-compose up -d

## Step 2:

Launch the Console using - http://localhost:4444/grid/console
